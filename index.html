In this iteration of my personal website, I was so inspired by Chris Krycho's wonderful Sympolymathesy that I determined to set up something very similar on my own, basically copying Chris's structural and stylistic decisions. I got as far as setting up a microblogging website similar to Chris's old microblog, but quickly lost steam. Clearly, copying someone else's blog structure built up over years of iteration and customization and expecting it to just work (and also attempting to build up my nascent blogging habit at the same time) wasn't a great idea. In any case, check out v0.manankhattar.com for a previous version of this website from another lifetime, and manankhattar.com for the latest version!
